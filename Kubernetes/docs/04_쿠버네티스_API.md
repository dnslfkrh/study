# 쿠버네티스 API

쿠버네티스 API를 사용하면 쿠버네티스 오브젝트들의 상태를 쿼리, 조작할 수 있다.

쿠버네티스 컨트롤 플레인의 핵심은 `API 서버`와 `그것이 노출하는 `HTTP API`이다. 사용자와 클러스터의 다른 부분과 모든 외부 컴포넌트는 API 서버를 통해 통신한다.

## 조작 가능한 쿠버네티스 API 오브젝트

- 파드 (Pod)
- 네임스페이스 (Namespace)
- 컨피그맵 (ConfigMap)
- 이벤트 (Event)

* 대부분의 작업은 `kubectl` 명령 등으로 수행할 수 있다. 또한 REST 호출을 통해서도 직접 접근이 가능하다.
